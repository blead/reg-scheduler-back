this is the default landing page for reg-scheduler-back.
this page contains information about API endpoints.

API endpoints

  • GET /courselist
    returns the result of querying the reg with given parameters

    url parameters

      courseId Course Id
      courseName Course Name

  • GET /courseinfo
    returns information about the course with courseId.
    result is null if the specified courseId is invalid.

    url parameters

      courseId Course Id (required)
